# Zetz-Tools-V1-py

Zetz Toolz V1 untuk mempermudah pengguna dalam melakukan sesuatu, script ini 100% tidak bayar dan bersifat NO ENC dan memakai bahasa pemograman yaitu Python

# Cara menjalankan script ini lewat Windows adalah:
1. Download Python (disarankan versi terbaru)
2. Jika sudah, setup Python
3. Buka terminal yaitu Win + R dan ketik CMD
4. Kalau sudah ketik cd C:\Users\NamaPengguna\Downloads\Zetz-Tools-V1-py-main
5. Kalau sudah maka ketik py dimztools.py atau python dimztools.py atau dimztools.py

# Cara menjalankan script ini lewat Android adalah:
1. Download "Termux"
2. Buka dan ketik "pkg install python"
3. Kalian tinggal Y/n da tunggu
4. Kalau sudah ketik "pip install request" dan "pip install colorama"
5. Ketik "pkg install git -y"
6. Ketik "git clone https://github.com/Zetz-dev/Zetz-Tools-V1-py](https://github.com/Zetz-dev/Zetz-Tools-V1-py.git"
7. Ketik "cd Zetz-Tools-V1-py"
8. Ketik "python dimztools.py"
